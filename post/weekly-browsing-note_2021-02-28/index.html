Deno を使って Vim/Neovim のプラグインを書く by denops.vim

良いですね。暇になったら触ってみたいです。

denops.vim は JavaScript/TypeScript のランタイムである Deno を利用して Vim/Neovim 双方で動作するプラグインを作るためのエコシステムです。以下のような特徴があります。

Vim / Neovim で同一コードを利用可能 Vim プラグインとしてインストールが可能 Vim script と比較してエンジンが爆速なのでゴリ押しが可能 ユーザーによるライブラリの依存管理が不要 プラグインが別プロセスとして動作するため Vim が固まりにくい プラグイン毎にスレッドが分かれているため相互干渉が起こりにくい プラグイン毎にパーミッションを設定可能なため安全性が高い（未実装）
